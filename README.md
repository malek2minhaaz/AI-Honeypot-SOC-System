# AI Honeypot SOC Dashboard

## Features
- Flask backend
- Honeypot login page
- SOC dashboard
- AI threat analysis
- Attack charts
- SQL injection detection
- Brute-force detection
- AI SOC assistant using Gemini

## Run Project

Install dependencies:

pip install -r requirements.txt

Run app:

python app.py

Open browser:

http://127.0.0.1:5000
